## 4.2 各工具详细介绍

### Cursor —— 新手首选

> **定位**：AI 编程鼻祖，社区最活跃

**优点**：
- 社区教程最多，解决问题快
- Model Auto 自动选择最强模型
- 集成 Git，操作简单
- 免费版本够用

**缺点**：
- 高级功能需要付费
- 国内访问稍慢

**适合**：
- 零基础新手
- 想快速出成果的人
- 所有场景

**官网**：https://cursor.com/

---

### Antigravity —— 前端神器

> **定位**：Gemini 3 Pro 前端最强

**优点**：
- 前端生成质量最高
- 免费额度充足
- 自动部署到 Vercel
- 实时预览

**缺点**：
- 目前主要是前端
- 需要 Google 账号

**适合**：
- 想快速看到 UI
- 做网页、小工具
- 前端开发者

**官网**：https://antigravity.new/

---

### Claude Code —— 复杂项目

> **定位**：代码质量顶尖

**优点**：
- 注释最详细
- 像暖男工程师
- 架构能力最强
- 上下文最长

**缺点**：
- 需要付费（但值）
- 国内访问稍慢

**适合**：
- 复杂项目
- 需要架构设计
- 长期项目

**官网**：https://claude.com/code

---

### Gemini 3 Pro —— 日常免费

> **定位**：免费额度最香

**优点**：
- 学生认证送 Pro
- 说人话，容易懂
- 上下文长
- 免费额度多

**缺点**：
- 代码质量不如 Claude
- 有时会"偷懒"

**适合**：
- 学生
- 日常 Q&A
- 预算有限的人

**官网**：https://gemini.google.com/

---

### VS Code + OpenRouter —— 预算党 0 元

> **定位**：完全免费

**优点**：
- 一分钱不花
- 可选多种模型
- 完全自定义

**缺点**：
- 配置麻烦
- 需要自己维护

**适合**：
- 极致省钱
- 开发者
- 喜欢折腾的人

**配置方式**：VS Code 安装 Continue 插件 + OpenRouter 免费 API

## 4.4 supermao 的"一贵一贱"配置

> **核心观点**：贵的模型干高价值思考，便宜的模型干机械劳动

### 日常配置

| 模型 | 用途 | 工具 |
|------|------|------|
| **贵** | 架构、决策、Debug | Claude Opus 4.5 |
| **便宜** | 日常生成、抓数据 | Gemini 3 Pro |

### 具体分配

```
项目开发：
- 规划架构 → Claude（贵）
- 写核心代码 → Claude（贵）
- 写重复代码 → Gemini（便宜）
- 修 Bug → Claude（贵）
- 抓数据 → Gemini（便宜）
- 写文档 → Gemini（便宜）
```
