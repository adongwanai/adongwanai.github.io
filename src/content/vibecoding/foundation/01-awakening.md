## 1.2 什么是 Vibe Coding？

> **定义**：Vibe Coding = 用自然语言"指挥"AI 生成代码，人只负责"说清楚要什么"。

### 核心区别

| 模式 | 做什么 | 需要什么 |
|------|--------|---------|
| **传统编程** | 手动写代码 | 懂语法、会 debug |
| **Vibe Coding** | 描述需求 | 会说话、会判断 |

### 你的新身份：超级个体 (Super Individual)

> **@supermao 金句**："技术是唯一稳定高溢价赛道，而 Vibe Coding 是 2026 年进入这条赛道的最低门槛。"

你会从"打工仔"变成"指挥官"：

- 过去：自己写代码 → 自己是程序员
- 现在：指挥 AI 写代码 → 自己是产品经理 + 架构师

## 1.4 核心概念：Agentic Workflow

> **2026 年最新趋势**：不只是"问一句答一句"，而是让 AI 自动完成整个工作流。

### 什么是 Agentic Workflow？

**简单理解**：你告诉 AI 一个目标，它自己分步骤完成。

| 传统模式 | Agentic 模式 |
|----------|--------------|
| 你说一句，AI 答一句 | 你说目标，AI 自己跑 |
| 每次都要手动 | 自动循环执行 |
| 单点能力 | 团队协作 |

### supermao 的"一贵一贱"配置

来自 @supermao《超级个体 Vibe Coding 行动建议（2025年12月版）》：

> **贵的模型**（Claude Opus 4.5 / GPT-5.1-Thinking）：做架构、决策、Debug
>
> **便宜的模型**（Gemini 3 Pro / 开源）：日常生成、抓数据、机械劳动

这就像一个公司：
- CEO（贵模型）做决策
- 员工（便宜模型）执行

## 1.6 动手实践：10 分钟做出你的第一个网页

> **这一步很重要！** 请务必动手跟着做。

### 步骤 1：下载 Cursor

访问 https://cursor.com/ ，下载并安装。

### 步骤 2：新建项目

1. 打开 Cursor
2. 点击 "New Project"
3. 创建一个空文件夹，取名 `my-first-web`

### 步骤 3：告诉 AI 你想要什么

在右侧聊天框输入：

```
帮我做一个简单网页：
- 标题是"我的第一个网页"
- 有一个输入框，可以输入名字
- 点击按钮后，显示"你好，{名字}！欢迎来到 Vibe Coding 世界"
- 用 Tailwind CSS 美化，暗黑模式
```

### 步骤 4：运行看结果

1. 点击 AI 生成的 "Preview" 按钮
2. 在网页输入你的名字
3. 点击按钮，看到效果

**恭喜！你已经完成了第一个 AI 网页！**
