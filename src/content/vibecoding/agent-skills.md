Agent Skills 是 AI 助手能力的延伸。这里收集了业内优秀的开源技能库和工具合集。

## 技能聚合项目

### [Baoyu Skills](https://github.com/JimLiu/baoyu-skills/blob/main/README.zh.md)
宝玉老师分享的 Claude Code 技能集，主要用于提升日常工作流中的效率，非常实用。

### [Awesome Claude Skills](https://github.com/ComposioHQ/awesome-claude-skills)
全面的 Claude Skills 资源大全，宣称支持 500+ 工具接入。

### [Anthropic Agent Skills](https://github.com/anthropics/skills)
Anthropic 官方维护的开源 Agent Skills 知识库，提供 16+ 核心技能示例，是学习编写自定义技能的最佳官方范本。

### [Awesome Vibe Coding](https://awesome-vibe-coding.com/)
专注工具导航，客观对比各类 AI 编程工具的优缺点，让你避免在选择工具时踩坑。
